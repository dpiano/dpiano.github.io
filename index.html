Website Process

Atom

Atom is the software program we use to code web pages in various languages like HTML5, CSS and Javascript. It has many features to make coding more efficient and less prone to errors.

GitHub

GitHub is an online developer's portal where we will be repositing our code. The code from Atom, once saved to the local folder, can be pushed to our repository through a desktop application called Git for Desktop.

Git for Desktop

This is the software that works to sync our web projects between a local folder and the online repository. It is important to never make changes from the repository itself, but rather directly from Atom into the local folder.

Git Page

This is a live version of your current web project based on the repository. To ensure it is up to date, you must sync it with your local folder.
